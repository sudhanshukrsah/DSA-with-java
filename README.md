## How to Run

1. Open a terminal and navigate to the project directory:
   ```powershell
   cd "e:\MY SIDE\JAVA ZONE\JAVA CODE\java-day2\src"
   ```
2. Compile the Java file:
   ```powershell
   javac Main.java
   ```
3. Run the program:
   ```powershell
   java Main
   ```
4. Follow the prompts to enter a string and a number.
